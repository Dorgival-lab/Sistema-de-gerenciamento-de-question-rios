Sistema de gerenciamento de questionários

Descrição: 
O sistema é totalmente desenvolvido em Java core e usa abordagem de questões de múltipla escolha. É um aplicativo baseado em menu e executa a operação com base na escolha do usuário a ser inserida diretamente no console.
Este programa usa um arquivo de texto simples para armazenar e manipular perguntas. As perguntas são armazenadas no formato CSV (Comma Separated Value). Ele mantém uma estrutura de dados muito simples. A primeira parte destinada à pergunta, as próximas quatro seções são quatro opções a serem fornecidas no console (na frente do usuário) e a última parte tem a resposta correta armazenada (aponta para o número do elemento na matriz de respostas). O arquivo "Questions.Txt" também está incluído no zip. Qualquer pessoa pode usar o arquivo ou criar novas ou modificar suas próprias perguntas. A apresentação das perguntas é totalmente dinâmica. A apresentação depende totalmente do arquivo mestre (que contém os dados das perguntas armazenados). Este arquivo será criado automaticamente pelo programa se não for encontrado no caminho do aplicativo. Ele também exibe o resumo do concurso junto com as respostas corretas de todas as perguntas respondidas incorretamente no final do questionário. Ele tem o foco preciso no número de perguntas apresentadas, respondidas ou ignoradas. O programa também calcula a porcentagem de precisão com base no nº. de respostas corretas. 

Pessoalmente, acho que este código é um ótimo exemplo de Programação Orientada a Objetos (OOPs).
